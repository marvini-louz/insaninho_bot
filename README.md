#pt-br

Insaninho é um bot de Discord simples e direto, feito para gerenciar cargos por reações. Ele envia mensagens personalizadas, adiciona os emojis configurados e atribui automaticamente os cargos aos membros que reagirem.

- Você pode usar ele em qualquer servidor, basta solicitar seu token no Discord Developer Portal e inserir na última linha do código.
- Outra configuração necessária é inserir o ID do canal em que o BOT fará a observação dos usuários que irão reagir para conseguir seus cargos.
- Para configurar os cargos e as reções basta usar !reactionrole. 
  - Exemplo: !reactionrole Escolha os seus cargos reagindo abaixo: "EMOJI" @"CARGO_DO_SERVIDOR".
- Para que os usuários saibam qual emoji escolher, é interessante você escrever uma mensagem antes, explicando cada cargo. (ou se desejar, edite o código e me mande uma pull request.)

#en-us

Insaninho is a simple and straightforward Discord bot designed to manage roles based on reactions. It sends personalized messages, adds configured emojis, and automatically assigns roles to members who react.

- You can use it on any server; simply request your token from the Discord Developer Portal and insert it in the last line of the code.
- Another necessary configuration is inserting the channel ID where the bot will monitor users who react to receive their roles.
- To configure roles and reactions, simply use !reactionrole.
  - Example: !reactionrole Choose your roles by reacting below: "EMOJI" @"SERVER_ROLE".
- To help users know which emoji to choose, it's helpful to write a message beforehand explaining each role. (Or, if you prefer, edit the code and send me a pull request.)
